== Description ==

This plugin provides functionality to add simple alert for pages, posts & custom post types.
Admin can manage alert text & alert pages, posts & custom post types from backend.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `simple-alert` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
